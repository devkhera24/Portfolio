ETL Pipeline

A soon ready to publish project with high benchmarks in which I am closely working on how to handle unstructured multimodal data inputs. The objective is to successfully extract entities of any domain and the relation between them. This will help in our collaborative projective of A vector + graph native database for RAG systems. This will improve the efficiency of RAG system by giving them clean structured inputs. The pipleine could also work as a standalone project for GenAI companies to get the desired data from a large set of unstructured data for training models with integrations like VQA and high quality OCR

Synthetic Data Generation

Won hackathon and pitched to Campus Fund. Our system takes raw data, including unstructured files and small datasets, and converts it into structured, AI-ready formats. At the same time, we expand the data using high-quality synthetic generation, while ensuring no personal data is exposed. We have proved our work based on 5 factor structural check F1 weights, and 0 PII detection.

3

Resolvr

Won hackathon and pitched to Mela VC. Built an AI-powered customer support automation engine for quick-commerce platforms (Zepto, Blinkit) using a multi-agent YAML-driven orchestration system. Engineered 18 ticket-routing workflows with LLM classification, human approval gates, persistent memory, and a real-time WebSocket operator dashboard.

4

SpindleFlow

Won 1st place in hackathon. Built a declarative YAML-driven multi-agent AI orchestration engine supporting sequential, parallel, and hierarchical sub-agent workflows with real-time web dashboard visualization. Integrated persistent vector memory via Pinecone, local sentence transformers for zero-cost embeddings, MCP tool support, and multi-provider LLM compatibility (OpenAI, Gemini).

5

IntelliDSA

Built a full-stack AI-powered mock interview platform using React, Firebase, and OpenAI GPT-4o that delivers real-time coaching nudges while candidates code, tracking hesitation patterns via keystroke heatmaps. Features automated session postmortems with multi-dimensional scoring (approach, complexity, edge cases, communication) and a patterns dashboard to surface recurring weaknesses across sessions.
